# Task Tracker — Full-Stack (Spring Boot + Angular)

🚧 In progress — building a multi-user task/issue tracker to practice full-stack development with Java/Spring Boot and Angular.

## Planned features
- JWT authentication (register/login, roles)
- Projects → Tasks → Comments
- Search, filter, pagination
- File attachments
- Email notifications
- Dockerized, CI-tested

## Tech stack
- **Backend**: Spring Boot, Spring Security, JPA/Hibernate, PostgreSQL
- **Frontend**: Angular, RxJS, Reactive Forms
- **Tooling**: Docker, GitHub Actions

## Status
Backend and frontend skeletons initialized. Feature work starting.
